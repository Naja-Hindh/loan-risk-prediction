# 💳 Credit Risk Prediction System

## 📖 Project Overview
This project focuses on analyzing real-world lending data to understand factors affecting loan default and to build a machine learning model for predicting credit risk.

The goal is to assist financial institutions in making data-driven decisions for loan approvals.

---

## 🎯 Objectives
- Analyze borrower financial data
- Identify key factors influencing loan default
- Build predictive models for risk assessment
- Classify customers into risk categories

---

## 📊 Dataset
- Source: Lending Club Loan Data (Kaggle)
- Due to size limitations, a sample dataset is used in this repository

🔗 Full dataset:
https://www.kaggle.com/datasets/adarshsng/lending-club-loan-data-csv
